Você está desenvolvendo uma aplicação full-stack chamada "Validador Avançado".

## Objetivo:
- Receber ideias de negócio e gerar análises estruturadas usando 
- arquitetura multi-agente com OpenAI.

## Stack obrigatória:
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- ORM: Prisma
- Banco: SQLite
- Testes: Jest + Supertest
- Auth: JWT

## Arquitetura:
- Orchestrator chama 3 agentes em paralelo
- Cada agente retorna JSON estruturado
- Resultado consolidado é persistido
- Frontend exibe dashboard

## Regras:
- Código limpo
- Funções pequenas
- Responsabilidade única
- Sem lógica duplicada
- Tipagem consistente
- Tratamento de erro obrigatório
- Respostas da IA sempre em JSON válido

## Objetivo técnico:
- Construir um projeto profissional, modular e escalável.
